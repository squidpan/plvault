# WHTD


## vi

- Type the start line number, a comma, the end line number, and **`d`** (e.g., `:10,20d` to delete lines 10 through 20 inclusive).
- 
- **Delete from current line to line m**: Type `:.,md` (where `.` represents your current cursor position).

- **Marking a range**: Go to line n, press **`ma`** to set a marker named `a`. Then go to line m and type **`d'a`** to delete everything in between

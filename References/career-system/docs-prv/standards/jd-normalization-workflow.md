# JD Normalization Workflow

## Principle

Web Clipper captures. Career System normalizes.

## Run folder

Each run uses:

```text
ops/runs/<run_id>/
├── input/
├── output/
├── logs/
└── run-manifest.md
```

## Raw JD

Raw captured files live in:

```text
data/jds/raw/
```

## Normalized JD

Normalized files live in:

```text
data/jds/normalized/
```

# Resume Finalization Standard v0.5.0

## Purpose

Final resumes should be clean markdown artifacts suitable for manual review, Obsidian, Teal import, or PDF export.

## Rules

- Remove generator comments.
- Remove internal review notes.
- Preserve truthful source bullets.
- Preserve employer/date/role structure.
- Do not enforce two-page output in v0.5.0.
- Keep summaries concise and skills sections readable.

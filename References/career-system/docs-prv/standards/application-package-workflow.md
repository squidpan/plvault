# Application Package Workflow

## Purpose

Create one folder per job application containing the working artifacts needed to apply.

## Inputs

```text
data/jds/normalized/*.md
data/roles/*.md
data/resume-versions/generated/*.md
```

## Output

```text
data/application-packages/application-<slug>-v1/
```

## Files

```text
jd.md
role.md
resume.md
cover-letter.md
application-manifest.json
```

The generated cover letter is a structured draft placeholder, not the final polished letter.

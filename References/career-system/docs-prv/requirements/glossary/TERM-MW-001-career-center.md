# TERM-MW-001 Career Center

## Definition

Career Center is the Motorweb application/platform that exposes Career System data through SQL, REST APIs, dashboards, command line workflows, and a future React UI.

## Purpose

Career Center makes Career System data operational, queryable, and presentable.

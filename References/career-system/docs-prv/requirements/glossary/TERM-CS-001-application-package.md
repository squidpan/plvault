# TERM-CS-001 Application Package

## Definition

An Application Package is the complete set of artifacts used to apply for a specific job opportunity.

## Includes

- tailored resume
- cover letter, if applicable
- source job description
- normalized job description
- role metadata
- gap analysis
- interview prep notes
- application status
- supporting notes or evidence

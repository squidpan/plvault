# EPIC-MW-002 Enterprise Platform Architecture

Status: Proposed

## Purpose

Define the long-term enterprise-style architecture for Motorweb and Career System, including service accounts, deployment topology, API gateway concepts, streaming data services, and production-style operations.

## Business Value

This epic captures future architecture ideas so they are not lost while keeping current implementation focused.

## Scope

- Linux service account standards
- API service execution model
- Data loader services
- API Gateway learning
- Public market data reference implementation
- Deployment promotion model
- Enterprise reference architecture

## Out of Scope

- Immediate implementation
- Production deployment
- Cloud deployment
- Kubernetes implementation

## Notes

These stories are backlog architecture items. They should guide future implementation but should not distract from current Career System operational use.

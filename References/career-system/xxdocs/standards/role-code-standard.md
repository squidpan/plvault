# Role Code Standard

Role code is generated from:

```yaml
role_family:
role_level:
role_qualifiers: []
```

Examples:

```text
ba-ai
sba-ai
sba-appsupport
sba-appsupport-ai-pm
sre
```

## Current Codes

| Code | Meaning |
|---|---|
| ba | Business Analyst |
| sba | Senior Business Analyst |
| bsa | Business Systems Analyst |
| sre | Site Reliability Engineer |
| ai | Artificial Intelligence |
| appsupport | Application Support |
| pm | Project Management |
| delivery | Enterprise Delivery |
| ops | Operations |
| infra | Infrastructure |
| cloud | Cloud |
| product | Product |

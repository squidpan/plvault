# Communications Workflow

Track three communication types:

```text
cover_letters
thank_you_letters
follow_ups
```

## Cover letters

Maintain only two masters:

```text
cover-master-ba.md
cover-master-sre.md
```

Generate company-specific versions from these.

## Thank-you letters

Thank-you letters are event-specific and should be stored by company and date.

Example:

```text
data/thank-you-letters/aegis/thank-you-aegis-2026-05-18.md
```

## Rule

A communication becomes part of the job history once sent externally.

# Resume Bullet Ranking Standard v0.4.6.2

## Purpose

Rank existing resume bullets and classify them into action categories.

## Categories

- Promote: move up and lead with these.
- Keep: useful supporting bullets.
- Compress: shorten, merge, or move into sector summaries.
- Remove: remove or heavily compress for this target role.

## Design

v0.4.6.2 refines v0.4.6.1. It still ranks existing bullets only; it does not rewrite bullets or generate final resumes.

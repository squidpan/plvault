# TERM-CS-002 Gap Analysis

## Definition

Gap Analysis is the comparison between a job opportunity's requirements and Paul's qualifications, experience, evidence, and learning needs.

## Outputs

- fit score
- strengths
- partial matches
- gaps
- effort estimate
- application recommendation
- interview preparation focus areas
- possible Skills notes

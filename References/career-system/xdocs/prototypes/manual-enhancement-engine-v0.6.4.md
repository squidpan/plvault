# Manual Enhancement Engine v0.6.4

## Purpose

Capture human-reviewed resume improvements and preserve them as reusable Career System metadata.

## Problem

Generated resume artifacts are useful but still require human judgment.

Manual edits improve:

- role-specific wording
- recruiter readability
- job-specific emphasis
- domain terminology
- elimination of awkward deterministic rewrite output

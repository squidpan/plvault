# US-MW-008E Release Runbook

Status: Done

## User Story

As a Career Center operator,
I want an A-Z release runbook,
so that releases can be deployed, validated, and recovered repeatably.

## Acceptance Criteria

- Runbook documents setup prerequisites.
- Runbook documents deployment steps.
- Runbook documents loader steps.
- Runbook documents API startup.
- Runbook documents smoke testing.
- Runbook documents rollback/rebuild notes.

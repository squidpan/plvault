# US-MW-008C Automated API Deployment

Status: Done

## User Story

As a Career Center operator,
I want a repeatable API deployment script,
so that the FastAPI service can be installed and started consistently.

## Acceptance Criteria

- Script creates Python 3.13 virtual environment.
- Script installs service dependencies.
- Script validates required environment variables.
- Script starts the API service.
- Swagger UI is accessible.

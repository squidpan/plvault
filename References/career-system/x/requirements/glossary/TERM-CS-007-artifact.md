# TERM-CS-007 Artifact

## Definition

An artifact is a durable file, document, record, output, or generated object that supports the Career System or Motorweb workflow.

## Examples

- job description
- normalized JD
- resume version
- application package
- gap analysis report
- user story
- data model
- API requirement
- UAT script

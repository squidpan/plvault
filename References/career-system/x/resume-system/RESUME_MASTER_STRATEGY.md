# Resume Master Strategy

## Master BA

The BA master should be the main career inventory. It should include all credible BA, application support, delivery, SDLC, UAT, requirements, documentation, operational workflow, and regulated financial/insurance experience.

Best target positioning:

```text
Senior Business Analyst + Application Support + Release Coordination + Production Operations
```

## Master SRE

The SRE master should focus on hands-on operations and support positioning, without overstating current Python/OpenShift experience.

Best target positioning:

```text
Market Data Application Support + Linux Operations + Release Validation + Runbooks + Monitoring Readiness
```

## Do not over-fragment

Avoid creating separate masters for every role:

```text
Master PMO
Master Insurance BA
Master Technical BA
Master Delivery BA
Master Product Owner
```

Instead, use the BA master and generate role-specific versions.

# Resume to Job Workflow

## Step 1 — Store JD

Save raw JD text under the job role record.

## Step 2 — Identify target lane

```text
BA / Application Support / Delivery → master-ba-resume.md
SRE / Production Support / Ops → master-sre-resume.md
```

## Step 3 — Create role-specific resume

Copy the matching template:

```text
templates/resumes/resume-role-version-template.md
```

## Step 4 — Tailor only these sections first

1. Title line
2. Professional summary
3. Top 8–12 FRBNY bullets
4. Skills section

## Step 5 — Transfer to Teal

Use markdown as the controlled source, then paste into Teal for formatting and export.

## Step 6 — Save exported PDF

Store PDF in the job/company folder and record the version.
